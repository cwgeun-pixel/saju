# 운세 해석 지침서

> 생년월일시 입력 → 사주팔자 · 자미두수 · 점성학 세 시스템에서 뽑아낼 수 있는 운세 정보 전체 목록

---

## 입력값

| 항목 | 내용 |
|------|------|
| 생년월일 | 양력 YYYY-MM-DD |
| 생시 | 시:분 (모를 경우 정오 기본값) |
| 성별 | 남 / 여 (자미두수 궁위 계산에 영향) |
| 출생지 | 도시 (위도/경도/시간대 자동계산, 한국 DST 포함) |

## 1. 사주팔자 (四柱八字)

### 기본 구조

- **4개 기둥** (年·月·日·時柱) 각각 天干 + 地支 = 8글자

### 뽑아낼 수 있는 정보

#### ① 오행 분포
- 木·火·土·金·水 각각의 강약과 과부족
- 용신(用神) / 기신(忌神) 판별 근거

#### ② 干支 관계
- 합(合), 충(沖), 해(害), 형(刑), 파(破)
- 기둥 간 상호작용으로 인생 굴곡 예측

#### ③ 장간 (藏干)
- 각 地支 속 숨겨진 천간
- 내면 성격과 인간관계 세부 분석

#### ④ 12신살 (神殺)
- 驛馬殺(역마), 桃花殺(도화), 劫殺(겁살) 등
- 인생 변동 패턴, 직업 특성, 이성운 키워드

#### ⑤ 12운성 (運星)
- 長生~絶까지 12단계 생명력 사이클
- 각 기둥·오행의 활성도 측정

#### ⑥ 납음 (納音)
- 60갑자 조합별 5행 분류
- 성향·직업·재물 보조 해석

#### ⑦ 좌법 (坐法) / 종격 (從格)
- 일간의 사회적 역할과 위상
- 종격(극단 구조)일 때 특수 운명 경로

#### ⑧ 대운 (大運) — 10년 주기
- 출생부터 130세까지 10년 단위 인생 흐름
- 각 대운 기둥과 원국의 합충 상호작용

#### ⑨ 세운 (歲運) — 연도별
- 매년 干支와 원국·대운의 충합 분석
- 해당 연도 길흉 포인트

### 궁합 (四柱 相性) — 두 사람 비교

> 두 사람의 사주를 교차 분석하여 관계의 조화와 긴장을 파악한다.

#### 궁합 입력값

| 항목 | 비고 |
|------|------|
| 두 사람 각각의 생년월일시 | 시간 미상 시 정오(12시) 사용 |
| 성별 | 대운 방향에 영향 (궁합 자체는 영향 적음) |
| 출생지 | 한국 사주의 경우 KST 기본 적용 |

#### 분석 항목

**① 일주 궁합 (日柱 — 가장 중요)**

| 레이어 | 분석 내용 |
|--------|-----------|
| 天干 (일간) | 합(合) — 甲己·乙庚·丙辛·丁壬·戊癸 다섯 쌍 |
| 天干 (일간) | 충(沖) — 甲庚·乙辛·丙壬·丁癸 네 쌍 |
| 地支 (일지) | 합(合), 반합(半合), 충(沖), 파(破), 해(害), 형(刑), 원진(怨嗔), 귀문(鬼門) |

**② 연주 궁합 (年柱 — 띠 궁합)**

| 관계 | 의미 |
|------|------|
| 地支 합(合) | 삼합·육합·방합 — 기본적으로 좋은 인연 |
| 地支 충(沖) | 子午·丑未·寅申·卯酉·辰戌·巳亥 — 갈등 소지 |
| 地支 원진(怨嗔) | 子未·丑午·寅酉·卯申·辰亥·巳戌 — 감정 충돌 |

**③ 전체 기둥 합충표 (4×4)**

두 사람의 각 4개 기둥을 교차 비교(16가지 조합)하여:
- 합(合) 횟수 → 높을수록 궁합 좋음
- 충(沖) 횟수 → 강한 긴장과 자극
- 원진(怨嗔) 횟수 → 감정적 상처 위험
- 귀문(鬼門) → 신비롭고 불안정한 연결

**④ 오행 상생상극 (일간 기준)**

| 관계 | 내용 | 궁합 의미 |
|------|------|-----------|
| 상생(生) | 木→火→土→金→水→木 | 한쪽이 다른 쪽을 살려주는 관계. 의존성 주의 |
| 상극(克) | 木→土→水→火→金→木 | 한쪽이 다른 쪽을 눌러주는 관계. 리더십 또는 갈등 |
| 비겁(比) | 같은 오행 | 경쟁하거나 돕는 동료형 관계 |

**⑤ 특수 관계**

| 살(殺) | 의미 | 궁합 해석 방향 |
|--------|------|----------------|
| 원진(怨嗔) | 서로의 기운을 억누름 | 초반 매력 → 시간 지날수록 갈등 누적 |
| 귀문관살(鬼門關殺) | 서로를 혼란스럽게 함 | 신비로운 끌림, 정신적 불안정 위험 |
| 도화살(桃花殺) | 이성 매력 | 상대방 사주에 내 일지·연지 도화가 있을 때 강한 이성적 매력 |
| 천을귀인(天乙貴人) | 귀인 관계 | 상대방이 나의 귀인 역할, 서로 도움 |

#### 관계 기호 빠른 참조

| 기호 | 한자 | 의미 | 궁합 점수 |
|------|------|------|-----------|
| 합 | 合 | 결합, 조화 | ++ |
| 반합 | 半合 | 부분 결합 | + |
| 충 | 沖 | 충돌, 자극 | -- (단, 자극적 관계) |
| 파 | 破 | 균열 | - |
| 해 | 害 | 상해 | - |
| 형 | 刑 | 제재 | -- |
| 원진 | 怨嗔 | 원망, 분노 | --- |
| 귀문 | 鬼門 | 귀신문 | 중립 (불안정) |

## 2. 자미두수 (紫微斗數)

### 기본 구조

- **12궁(宮)** × **111개 이상 별** = 명반(命盤)

### 12궁 — 인생 전 영역

| 궁 | 영역 |
|----|------|
| 命宮 | 기본 성격, 운명, 수명 |
| 兄弟 | 형제자매, 우정, 동업 |
| 夫妻 | 결혼, 연애, 파트너십 |
| 子女 | 자녀, 창의성, 유산 |
| 財帛 | 재물, 수입, 소비 패턴 |
| 疾厄 | 건강, 질병 취약점, 체질 |
| 遷移 | 이사·해외, 외부 환경 |
| 交友 | 친구, 사회적 인맥 |
| 官祿 | 직업, 커리어, 명예 |
| 田宅 | 부동산, 가정환경, 상속 |
| 福德 | 행복감, 영적 복, 내면 평화 |
| 父母 | 부모 관계, 윗사람, 멘토 |

### 뽑아낼 수 있는 정보

#### ① 14개 주성 배치
- 紫微·天機·太陽·武曲·天同·廉貞·天府·太陰·貪狼·巨門·天相·天梁·七殺·破軍
- 어느 궁에 어떤 밝기로 안착했는지 → 해당 인생 영역의 근본 에너지

#### ② 사화 (四化)

| 화 | 의미 |
|----|------|
| 化祿 | 이득, 풍요 |
| 化權 | 권력, 주도권 |
| 化科 | 명예, 시험운 |
| 化忌 | 손실, 장애 |

- **본명반** / **대한(10년)** / **유년(流年)** / **유월(流月)** 4단계에서 각각 활성화
- 화기(化忌)가 어느 궁에 입궁하는지 → 해당 해 주의 영역 예측

#### ③ 대한 (大限) — 10년 주기
- 출생부터 130세+까지
- 각 대한 기간의 주관 궁위·주성·사화 분석

#### ④ 유년 (流年) — 연도별
- 해당 연도 干支와 궁위 이동
- 유년사화와 본명반 충돌·조화

#### ⑤ 유월 (流月) — 월별
- 해당 연도 12달 세부 흐름
- 월별 궁위 변화와 중요 사건 타이밍

#### ⑥ 오행국 (五行局)
- 水2局 / 木3局 / 金4局 / 土5局 / 火6局
- 대한 시작 나이와 전체 인생 에너지 결

## 3. 점성학 (西洋占星術 / Natal Chart)

### 기본 구조

- 10행성 + 4각도 + 12하우스 + 행성 간 상호각도 (Placidus 방식)

### 뽑아낼 수 있는 정보

#### ① 10행성의 하우스 + 별자리

| 행성 | 주관 영역 |
|------|-----------|
| ☀️ 태양 | 핵심 정체성, 의지, 활력 |
| ☽ 달 | 감정, 본능, 어머니 |
| ☿ 수성 | 사고, 소통, 학습 |
| ♀ 금성 | 사랑, 미적 감각, 가치관 |
| ♂ 화성 | 행동력, 욕망, 경쟁 |
| ♃ 목성 | 팽창, 행운, 지혜 |
| ♄ 토성 | 책임, 제한, 성숙 |
| ♅ 천왕성 | 혁신, 반란, 급변 |
| ♆ 해왕성 | 영성, 환상, 직관 |
| ♇ 명왕성 | 변환, 심층 권력, 재생 |

#### ② 4개 앵글 (Angle)

## 4. AI 종합 해석 8개 섹션

세 시스템의 계산 결과를 모두 입력받아 아래 8개 주제로 통합 해석한다.

| # | 섹션 | 핵심 질문 |
|---|------|-----------|
| 1 | 성격·기질 | 세 시스템의 성격 특성 교차 분석 — 공통 패턴은? |
| 2 | 사주 심층 | 팔자 기둥 간 합충·오행 불균형이 삶에 어떤 의미인가? |
| 3 | 자미두수 심층 | 12궁 주성 배치와 사화 활성화가 말하는 것은? |
| 4 | 점성학 심층 | 행성·하우스·상 패턴이 드러내는 삶의 주제는? |
| 5 | 연애·결혼 | 세 시스템의 인연 지표를 교차 비교하면? |
| 6 | 커리어·재물 | 적합 직업군과 재물 흐름의 공통 신호는? |
| 7 | 연간 운세 | 현재 연도 대운·유년·트랜짓 종합 전망 |
| 8 | 종합 통찰 | 동서양 세 시스템이 동시에 가리키는 핵심 메시지 |

## 5. 해석 시 유의사항

- **생시 불명확** → 자미두수 명궁 위치·사주 시주 모두 오차 발생. 시주 제외 7자 해석 권장.
- **음력 자동 변환** → 자미두수는 음력 기준이므로 양력 입력 후 내부 변환 필수.
- **출생지 정밀도** → 점성학 하우스 커스프와 ASC는 출생 도시의 정확한 경위도에 의존.
- **세 시스템 가중치** → 어느 하나를 정답으로 보지 말 것. 세 시스템이 동시에 지목하는 주제일수록 신뢰도 높음.
- **운의 방향성** → 사주·자미두수는 동양 음양오행 프레임, 점성학은 서양 상징 프레임. 번역 불가한 뉘앙스 차이 존재.
- **궁합 주의사항** → 원진·귀문이 있다고 무조건 나쁜 궁합이 아님. 충(沖)도 자극과 성장의 동력이 될 수 있음. 전체 흐름으로 판단.
